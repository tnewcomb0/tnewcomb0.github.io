Ruby Arrays and Hashes

Ruby Coding, Back-end, Ruby
0 Comments October 30th, 2015

Ruby has two similar elements, arrays and hashes, that at first glance look quite similar. Both contain multiple items, such as integers (whole numbers), floats (numbers with decimal points), and strings (words). The main difference, however, is that a hash contains paired values (such as a name paired with an address).

First, lets look at arrays. Arrays look like this:

random_array = [1, "cheese", "18.7", "panda"]

Basically, this is a list containing the integer 1, the string "cheese", the string "18.7", and the string "panda". Arrays are constructed so that you can access the values or items in them with a cooresponding index number. An index number is just a value that is automatically assigned, by Ruby, to each item in an array. So, in our example array: 1 would be assigned the index 0, "cheese" would be assigned the index 1, "18.7" would be assigned the index 2, and "panda" would be assigned the index 3. To access one of them you would use the command random_array[2], and that would give you "18.7". You may notice that the index does not start at one, rather it starts at 0. Arrays have many uses, and are used everywhere in Ruby.


Next, hashes are a close cousin to arrays. Instead of using an automatically assigned index however, hashes get both their index number and its value assigned by the coder. It also goes by slightly different syntax. You call them keyes and values, so a key/value pair. A hash would look like this:


random_hash = { "twenty" => 20, "thirty" => 30 }

Obviously this wouldn't be a very useful hash, but it demonstrates how they work well. So, the key "twenty" has its value of 20, and the key "thirty" has its value of 30. Hashes are a nice alternative because you can store data that you want to be paired with something else. A few examples of this would be usernames and passwords, names and phone numbers, and widths and heights. It provides a quick and easy way to store information that you want kept together.
