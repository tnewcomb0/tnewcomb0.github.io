The .inject Method

The enumerable module in Ruby puts many new methods at your disposal. It was pretty hard to choose from them, as many of them seem really useful. I decided to wrrite on one that may not see as much use as some of the others, but it still will be quite useful in certain situations. That method is called .inject.

The .inject method is used, in its simplest form, to input something between each element in an array. So, for example, if you wanted to multiply all the numbers in an array together, you could write something like this:

def multiply_me(array) x = array.inject(:*) p x end multiply_me([2,3,4]) >>24

Now, let's walk through this step by step. First, we are defining a method called multiply_me and giving it a parameter of an array. We are then setting the new variable x equal to array.inject(:*), which will put a * (which is the multiplication symbol in Ruby) between each element in an array (we are also assuming that the array contains only numbers, otherwise we would have to include .to_i). Finally, .inject will also run the array with the injected symbol, so you will get a value returned, in this case 24, which is 2 * 3* 4.

This is only a very simple way to use .inject, it has many more uses beyond just putting in math symbols, but I'll leave you to look into those on your own!
